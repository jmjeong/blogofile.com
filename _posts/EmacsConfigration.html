현재 사용하고 있는 emacs configration이다. Github jmjeong-emacs에 최신버젼을 운영 중이다. 

git clone git://github.com/jmjeong/jmjeong-emacs.git

각 package 설정마다 .el을 분리하여 파일로 만들었고, init.el에서 필요 elisp을 읽는 구조로 변경했다.

auto-install/
auto-install.el 패키지에서 자동으로 install되는 파일이 저장되는 folder이다. Upgrade도 auto-install package에 의해 관리된다.
vendor/
여러 emacs site에서 받은 package들 저장. 단일 el파일만 저장하고, 나머지는 update.sh에 의해 자동으로 받아오는 구조로 관리하다가 여러 machine을 옮겨갔을때 설정하는 것이 번거로워서 local copy를 유지하는 방식으로 변경했다. update.sh파일을 통해서 필요한 파일은 자동 upgrade.

init.el 에서 elisp-root-dir를 download한 디렉토리로 변경하여 사용하면 된다. 

(defcustom elisp-root-dir "~/my-dot-emacs"
  "The root directory where elisp is installed")

GNU Emacs 23.1.50.1 (i386-apple-darwin9.8.0)에서 사용 중이다.
